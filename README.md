#Tease

### 1. Install Global Dependencies:

> #### [Node.js](https://nodejs.org/en/download/)

> #### Harp
    npm install -g harp

> #### Gulp:
    npm install -g gulp-cli

### 2. Install Project Dependencies:
>     cd "C:\Users\..."
    npm install --only=dev

### 3. Run Harp Server:
>     harp server

> Open browser and type `localhost:9000`

### 4. Compile Harp Site:
>     gulp
